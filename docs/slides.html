The Occupancy model for survey data

Basic idea: Data contain information about two processes:

  • The species occupancy (ecological process)

  • The detection (observation process)

Basic model form

  • Ecological process

    \[ z_i\sim\text{Bernoulli}(\psi_i) \]

  • Observation process

    \[ y_{ij}\mid z_i=z_i\sim\text{Bernoulli}(p_{ij}) \]

Note: The model requires multiple visits per site to be identifiable

Occupancy Models in INLA

For a set of set of \(M\) discrete sites, let the detection history of the \(i\)th site \(\mathbf{d}_i= y_{i1},\ldots,y_{iK}\).

The joint likelihood is then:

\[ \begin{aligned} L(\pmb{\psi},\textbf{p}|\mathbf{y}_1,\ldots,\mathbf{y}_M) &= \underbrace{\prod_{i=1}^{m} \left[\psi_i \prod_j^{K_i} p_{ij}^{y_{ij}} (1-p_{ij})^{1-y_{ij}}\right]}_{\text{sites with at least one detection}}\underbrace{\prod_{i=m+1}^M\left[\psi_i\prod_j^{K_i}(1-p_{ij})+(1-\psi_i)\right]}_{\text{sites with only 0's}}. \\ &= \prod_{i=1}^{M} \left[\psi_i \prod_j^{K_i} p_{ij}^{y_{ij}} (1-p_{ij})^{1-y_{ij}} + \mathbb{I}_{\sum_j y_{ij} = 0}(1-\psi_i) \right], \end{aligned} \]

The likelihood for occupancy model is factorized into univariate models for the marginal distribution of each site detection history.

\[ L(\pmb{\psi},\textbf{p}\|\mathbf{y}\_1,\ldots,\mathbf{y}*M) =* \prod{i=1}^{M} \left[\psi_i \prod_j^{K_i} p_{ij}^{y_{ij}} (1-p_{ij})^{1-y_{ij}} + \mathbb{I}_{\sum_j y_{ij} = 0}(1-\psi_i) \right] \]

Estimating Occupancy Models in INLA

Belmont et al. (2024) show how this a LGM (and thus possible to be fitted using INLA ) under these conditions:

  • Detection histories are conditional independent given the occurrence and detection probabilities.

  • Considering detection probabilities \(p\) as hyperparameters and link the linear predictor \(\eta_i\) to the probability of occurrence \(\psi_i\) via a known link function \(g(\cdot)\).

  • Assume a Gaussian prior for \(\eta_i\).

The linear predictor \(\eta_i\) can include covariates, smooth terms, random effects, etc. e.g., \[g(\psi_i)=\beta_0 +X'\beta + \omega(s)\]

While the detection model is compromised by a less complex structure where only a limited number of site or survey level covariates can be included, e.g., \[\text{logit}(p_{ij})= \gamma_0 + \gamma_1 \times w_{ij} + \ldots\]

\[ \lambda(s)=\exp{X(s)'\beta + Z(s)} \]

Thinned point process model

  • The true locations \(\mathbf{s}=s_1,\ldots,s_m\) where a species occur are a realization of a Point Process, specifically a log-Gaussian Cox process (LGCP).

  • The true point pattern is partially observed due to imperfect detection and accessibility constrains reflected by an uneven sampling effort.

Thinned point process model

  • The true locations \(\mathbf{s}=s_1,\ldots,s_m\) where a species occur are a realization of a Point Process, specifically a log-Gaussian Cox process (LGCP).

  • The true point pattern is partially observed due to imperfect detection and accessibility constrains reflected by an uneven sampling effort

  • The locations \(\mathbf{s}=s_1,\ldots,s_n\) where opportunistic encounters have actually occurred are the result of a thinned point process.

\[ \begin{aligned} \lambda^{pp}(s)&=\lambda(s)\times p(s)\\ \text{logit}(p(s)) &= \alpha^{pp}_0 + \alpha^{pp}_1\times g(s) + \ldots \end{aligned} \]

Estimation of the thinned point process model

The likelihood function for the thinned point process is given by:

\[ L_{thinn}(\beta,\alpha) = \exp\left\{\int_B\lambda(s) p(s) ds) \right\}\prod_i^m \lambda(s_i)p(s_i) \]

  • Simpson et al. (2016) proposed an approach to approximate both the stochastic integral and the evaluation of the field at the data points using INLA.

  • In R-INLA we can defined different model components as part of the additive predictor (e.g., the Stochastic Partial Differential Equation (SPDE) approach to a GRF with Matérn covariance).

  • inlabru allows the predictor to be a non-linear function of the parameter vector through an iterative model fitting scheme.

  • We can use inlabru fixed point iteration method to estimate thinning probabilities \(p(s)\).

  • The point process model proposed for CS data is defined on a continuous space, but the occupancy model for survey data is defined in a discretised space.

  • We can use the fact that the number of individuals in a bounded region can be characterized as \(N(A_i)\sim \mathrm{Poisson}(\int_{A_i}\lambda(s)~ds)\).

  • The probability that an areal unit \(A_i\) is occupied by a species is given by:

\[ \psi_i = \mathrm{Pr}\left[N(A_i)>0\right] = 1 - \exp\left(-\int{A_i}\lambda(s)~ds\right), \]

Thus, the intensity-based OM likelihood becomes:

\[ L(\psi_i,p_{ij}) = \left[\prod_i^N \left(1-\exp\left\{-\int_{A_i}\lambda(s) ds\right\}\right)\prod_j^{K_i}p_{ij}^{y_{ij}}(1-p_{ij})^{1-y_{ij}} \right] \times\\ \prod_{i=N+1}^M\left[ \left(1-\exp\left\{\int_{A_i}\lambda(s) ds\right\}\right) \prod_j^{K_i}(1-p_{ij})+\exp\left\{\int_{A_i} \lambda(s)ds\right\} \right]. \]

Case Study: Modelling the Distribution of the black-throated blue warbler in Pennsylvania, USA.

  • Occurrence data for the Black-throated blue warbler (Setophaga caerulescens) in Pennsylvania, USA are available from two sources:

  1. The Pennsylvania Breeding Bird Atlas (BBA): Point count surveys collected through a standardized data collection protocol from 2005 to 2008 .

  2. The second data set does not follow a standard data collection protocol, but instead are observations reported by citizen scientists to the eBird platform in 2008.

  • LDI is the land development index proposed by Brown and Vivas (2005). The smaller the index values is, the less the urban development is.
